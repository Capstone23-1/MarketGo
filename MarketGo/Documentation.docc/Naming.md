# 변수/파일명 정보 공유
*MVVM패턴 이용*

### MVVM (Model-View-ViewModel
- View
  - UI를 그리는 곳
- Model
  - 데이터를 저장 및 관리
- ViewModel
  - 뷰와 상호작용하는 메소드
  - 데이터를 처리하는 메소드
  - ViewModel은 View에서 받은 입력을 Model에 전달하여 데이터를 업데이트
