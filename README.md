## 핵심기능 소개
### 빠른 상품 등록
- OCR 인식 후 NLP 모델을 이용해 물건이름/가격/단위를 인식후 자동으로 물품등록창을 채워줍니다.
- 상인분들이 바쁜 와중에 물건을 손쉽게 등록할 수 있습니다.
